# Proyecto incubadora G1

Instrucciones para iniciar el proyecto:

1. Clonar o forkear el repositorio:

```bash
git clone https://github.com/dlab-team/g1-frontend
cd g1-frontend
```

2. Instalar las dependencias

```bash
npm install
```

3. Ejecución ▶️

```bash
npm run dev
```
